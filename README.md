# javascript_md5
使用javascript计算文件MD5，可以用在用户上传文件的时候，计算文件的MD5，让用户进行二次确认的目的


# 如何运行
该工程使用 django1.11 搭建

```
cd django1_11_project_test
python manage.py runserver
```
